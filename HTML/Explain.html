Transform Property

"The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements." Read More

Example

  • translate() - การย้าย element ไปตำแหน่งที่กำหนด
  • rotate() - การหมุน element
  • scale() - การย่อขยาย element
  • skewX() - ทำให้เอียงในแนวนอน
  • skewY() - ทำให้เอียงในแนวตั้ง
  • x() - เป็นการรวม transform ทั้งหมดยกเว้น rotate เข้าด้วยกันในบรรทัดเดียว โดยเขียนได้ดังนี้ matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())

Transitions Property

CSS transitions allows you to change property values smoothly, over a given duration.Read More

Example

  • transition-delay คือระยะเวลาเป็นตัวเลขวินาทีก่อนที่จะให้เริ่มการทำงาน
  • transition-duration คือระยะเวลาในการแสดงผลจากคุณลักษณะแรกไปยังคุณลักณะที่สอง สรุปช้าหรือเร็วสามารถกำหนดได้จาก Property นี้ได้เลยนั่นเอง
  • transition-property คือคุณสมบัติของ div หรือ element ที่เราต้องการให้เปลี่ยนแปลง เช่น background-color, color, border-color เป็นต้น
  • transition-timing-function คือรูปแบบหรือลักษณะในขณะที่กำลังเปลี่ยนแปลง ค่าเริ่มต้นจะเป็น ease และมีให้เลือกอีกมากมายเช่น linear, ease-in, ease-out, ease-in-out เป็นต้น
